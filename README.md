# le-yu-guide

## 项目简介

`le-yu-guide` 是一个用于归档和发布独立 HTML 页面的仓库。本仓库不针对任何特定域名或网站，仅作为静态页面内容的集中存放与展示平台。每个页面均为自包含的 HTML 文件，可直接在浏览器中打开查看。

## 目录结构

```
le-yu-guide/
├── pages/          # 存放所有 HTML 页面文件
│   ├── example1.html
│   └── example2.html
├── assets/         # 页面引用的静态资源（样式、脚本、图片等）
│   ├── css/
│   ├── js/
│   └── images/
└── README.md       # 本文件
```

- **pages/**：核心目录，所有独立 HTML 页面均放置于此。
- **assets/**：页面所需的辅助资源文件，按类型分目录存放。

## 页面归档说明

- 每个 HTML 页面均为独立、完整的文档，不依赖外部服务或动态后端。
- 页面内容可能涉及信息展示、工具、笔记等，具体用途以页面自身说明为准。
- 归档遵循“一次发布，长期保留”原则，内容更新时以新版本文件替代旧文件，历史版本可追溯至 Git 提交记录。

## 如何贡献

欢迎提交改进或新增页面。请遵循以下流程：

1. Fork 本仓库。
2. 在 `pages/` 目录下创建或修改 HTML 文件。
3. 如有新增资源，请放入 `assets/` 对应子目录。
4. 确保页面在本地浏览器中能正常打开，无明显错误。
5. 提交 Pull Request，并简要说明变更内容。

## 维护说明

- 本仓库由维护者不定期更新，不保证频繁提交。
- 页面内容可能随时间调整或移除，但会尽量保持向后兼容。
- 如发现链接失效或页面显示异常，欢迎通过 Issue 反馈。
